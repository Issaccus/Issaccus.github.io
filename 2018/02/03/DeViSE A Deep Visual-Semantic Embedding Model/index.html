DeViSE: A Deep Visual-Semantic Embedding Model

Abstract

  在视觉识别任务中，当需要识别的分类很多时，训练数据（已标记的图片）的匮乏限制了识别系统的识别能力。本文利用已标记的图片数据（ labeled image data）和未注释文本中的语义信息（ semantic information gleaned from unannotated text）建立了深度视觉-语义嵌入模型（DeViSE），并在”1000-class ImageNet object recognition challenge”达到最优的性能，同时说明了可以通过充分利用语义信息来对训练数据集外的10,000个图像标签进行预测。

Introduction

  视觉世界由大量物体组成，而物体的标签是模糊的，对于不同的任务可能有不同的标签，或者可能存在多个标签。当前最先进的识别系统采用了严格定义一个小的标签类，并人为地把图像指派给不同标签的方法，根据标签建立已标记的训练数据集，识别系统即建立在一个N类的离散分类器上。因为N个标签是离散且无关的，识别系统就不能将和已有标签相关的语义信息转移到未曾见过的词语，一个解决办法就是不再人为地把图像划分到离散的类里，而是考虑视觉空间的自然连续性。
  DeViSE利用文本信息来学习标签之间的语义联系，并把图像映射到一个语义嵌入空间。DeviSE和目前最先进的识别系统有同级别的性能，并且可以利用视觉和语义之间的相似性来为未见过的分类预测其标签。

Proposed Approach

  方法的目标是利用文本域上学习到的语义信息，转移到一个训练好的识别系统上。首先预训练一个简单的神经语言模型来学习词语的向量表示，同时再预训练一个最先进的物体识别神经网络，连同传统的softmax输出层。然后使用softmax输出层的上一层，重新训练来预测语言模型得到的向量表示，由此建立一个深度的视觉-语义模型。

Language Model Pre-training

  The skip-gram text modeling architecture [1, 2]
  模型通过预测文档中每个词项的邻近词项将词项表示成嵌入向量。因为同义词倾向于出项在相似的语境下，用这个简单的目标函数驱使模型为语义相关的词语学习到相似的嵌入向量。
  数据集： 从wikipedia.org上抽取的5,700,000个文档，包含5,400,000,000个词语。网页上的文本被分割成若干个词语或词组，和常用的视觉物体识别数据集（ Imagenet: A large-scale hierarchical image database）组成了一个约155,000个词项的词典。

Visual Model Pre-training

  based on the winning model for the 1,000-class ImageNet Large Scale Visual Recognition Challenge (ILSVRC) 2012 [1]
  该模型作为性能比较的基准，同时也是视觉-语义嵌入模型的初始化之一。
  数据集：ILSVRC 2012 1K dataset
